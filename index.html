About

Ringo is a Python based high level web application framework build on top of Pyramid. The framework can be used to build form based management or administration software. Ringo is a littlebit like an administration application on stereoids for your data.

Strength

Collecting and managing data. Fast and flexible development because of the building block architectur and "Ready to Use" components.

Major field of application for Ringo are form based and data driven applications. Examples are: Case Management software in social casework, data aquisition and evaluation in a scientic environment, general colloborative work and management of your data.

Motivation

In the last 10 years I have build similar webapplication from scratch much too often! Although they were different in what they exactly did, they all shared a common set of needs like user administration, a permission system or nice forms with validation and so on.

This project started in 2013 as a private spare time project with the goal to create a tool which helps me to build such applications much faster than before. And it worked out!

Status

Ringo is under active development for about two years now and it is becoming more and more stable! The development currently happens exclusive on the head of the source repository and might involve changes which are not compatible with older versions. There is no maintainance of older versions of Ringo!

It is already in productional use for some customers so it is at least beta quality.

Features

Building block system

Write less code! The key idea behind ringo is to provide a building block system which allows you to easily add and link Modules which will handle data persistance and buisness logic. Adding modules is very easy and you will not even need to write a single line of code to get started to store data in your application.

Batteries included

Many of often used features in webapplication like Logging, Versioning, Comments, Fileuploads, Printtemplates, Tags, Nested Elements are already included in ringo. They can easily be added to your modules through mixins.

User management

Manage your Users in Usergroups and store Profiles along with them in the web interface. Optionally provide new user a Userregistration and allow users to reset their password using a email based Passwordreset function in case they forget their password.

Permission system

A fine graded role based permission system allows you define who is allowed to do what on each single item in your application.

Powerfull forms

In data driven application forms are one of the most importent elements in your appliction. Ringo uses the Formbar library, which gives to large freedom to design and validate your forms.

Workflows

A state machine in Ringo let you define the buisness logic in your application. Define actions when doing transations between states and ensure with conditions that a certain state can only reached if your defined preconditions are fullfilled.

Configurable

Changing the layout of the GUI? Changing the default behaviour of a particual modul action? Not problem. Ringo provides many ways to adapt the default application to your need. Further ringo come with a administraion UI which which let you configure many aspects of your application.

REST INTEFACE

A Ringo based application is not only traditional webapplications with an integrated GUI. Every module also has a REST-Interface for every core method which can be used from external clients.

Extensible

Using the plugin mechanism of Ringo developers have the option to write external extensions for Ringo which can be plugged into your application to add some general functionallity.

Get started!

The following steps will show you how easy it is to create a Ringo based application and start storing your data in a new modul.

1. Download and install ringo

Get the source from Bitbucket and install Ringo. I recommend you to to this in a virtialenv.

                  hg clone https://bitbucket.org/ti/ringo
                  cd ringo
                  python setup.py development

2. Create a new ringo based application

Ringo comes with an application scaffold which make it easy to create a new Ringo based application.

                  pcreate -t ringo Foo
                  cd foobar
                  python setup.py development

To be able to do the next steps you need to install your new application too.

3. Initialise the database

Initialise the database by invoking one of the nice administration commands.

                  foo-admin db init

Now the application is ready and could be started. But wait! We need a new modul to store your data.

4. Create a new modul for your application

Creating a new modul is easy. Thanks to the administration command its simply a question of doing the following.

                  foo-admin modul add bar

This command will create a new modul calle "bar" and also creates default LCRUDIE (List, Create, Read, Delete, Import and Export) views!

5. Start your new application

Here we are! Its time to lauch the application.

                  pserve --reload development
